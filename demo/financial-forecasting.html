Scenario Simulation

Simulates what-if scenarios against baseline forecasts, comparing cash positions and identifying impacts.

Input

Output

Baseline vs scenario, impact analysis, recommendation

Variance Analysis

Analyzes budget-to-forecast variances by category, identifies root causes, and recommends corrective actions.

Input

Output

Variance summary, category breakdown, recommendations

Output

Risk score (1–10), key risks, mitigation strategies
